# Body Count Tracker

A privacy-focused React Native app for tracking intimate encounters, built with Expo.

## Features

- 📊 Track encounters with detailed stats and analytics
- 👥 Partner management with ratings and notes
- 📅 Calendar view of your history
- 🏆 Achievements and milestones
- 🧪 STI test reminders
- 📥 Import/export data from Excel/CSV
- 🔒 Face ID/biometric protection
- 🌙 Dark mode interface

## Tech Stack

- **React Native** with Expo
- **SQLite** for local data storage
- **React Navigation** for routing
- **Victory Native** for charts
- **Skia** for advanced graphics

## Getting Started

### Prerequisites

- Node.js 18+
- iOS Simulator or Android Emulator
- Expo CLI

### Installation

```bash
npm install
```

### Running the App

For development build:
```bash
npx expo run:ios
# or
npx expo run:android
```

## Privacy

All data is stored locally on your device using SQLite. No data is ever sent to external servers.
